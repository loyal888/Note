#### HashTable
![](../../../image/2245e6a9.png)

- 初始容量：11，负载因子.75

- 实现方式：数组加链表

- 扩容方式： `int newCapacity = (oldCapacity << 1) + 1;`

- 数组下标：`int index = (e.hash & 0x7FFFFFFF) % newCapacity;`

- hash冲突采用头插法

- 同步的实现机制
